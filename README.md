# App del Clima - React + TypeScript

Aplicación para consultar el clima actual de algunas ciudades del mundo.  
Desarrollada con React, TypeScript y consumo de una API pública de clima.


# Tecnologías utilizadas
- React  
- TypeScript  
- Axios  
- Zod   
- Vite
